API

Edit API

Minimal video processing backend for uploads, trims, overlays, watermarks and transcodes. Built with FastAPI, FFmpeg, Postgres and background jobs that report progress clearly.
Designed as a compact service layer for editor tools and media workflows.

Pipeline Capabilities

What it handles

The API stores uploaded videos, creates trimmed clips, applies text, image and video overlays, adds full-duration watermarks, and generates processed variants for multiple resolutions.

Work is tracked as durable Postgres-backed jobs with statuses for pending, active, successful and failed operations, so client apps can poll progress without owning the processing lifecycle.
